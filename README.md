# Gutenburger Pesto

A series of free, open-source visual novels adapted from public domain works sourced through [Project Gutenberg](https://www.gutenberg.org).

**Goal: free on Steam (Mac/PC/Linux) and playable in-browser — no install required.**

> Early-stage WIP. Not yet published anywhere.

---

## Adaptations

### The House on the Borderland *(in development)*
*William Hope Hodgson, 1908*

A recluse living in an isolated house in rural Ireland begins to experience visions of an incomprehensible cosmic horror. Part journal, part fever dream.

[Source text on Project Gutenberg →](https://www.gutenberg.org/ebooks/10839)

---

## Running the project

1. Download [Godot 4.6](https://godotengine.org/download/)
2. Clone this repo
3. Open `theHouseOnTheBorderland/project.godot` in Godot
4. Hit **Play**

No additional dependencies or build steps required.

## Contributing

All contributions are welcome — code, art, writing, playtesting.

The project uses:
- **Godot 4.6** — GL Compatibility renderer (targets web/mobile)
- **GDScript** — no C# or plugins required to get started
- **JSON** for all narrative content — no engine knowledge needed to edit story beats
- **Blender / GLTF** for 3D assets — low-poly, PSX-adjacent style

Open an issue or a pull request. If you're tackling something substantial, open an issue first to align on approach.
